# File Search Tool

Программа для поиска текста в файлах с графическим интерфейсом Windows.

## Описание

File Search Tool — это Windows-приложение на языке C с использованием Win32 API, предназначенное для быстрого поиска ключевых слов в файлах указанной директории. Программа поддерживает рекурсивный поиск, фильтрацию по расширениям файлов и множество других настроек.

## Возможности

- ✅ Поиск нескольких ключевых слов одновременно
- ✅ Рекурсивный поиск во всех подпапках
- ✅ Фильтрация по расширениям файлов
- ✅ Поиск с учётом/без учёта регистра
- ✅ Поиск целых слов
- ✅ Отображение результатов с указанием номера строки
- ✅ Экспорт результатов в текстовый файл
- ✅ Поддержка кириллицы в именах файлов и тексте
- ✅ Прогресс-бар для отслеживания выполнения

## Требования

- Windows 7/8/10/11 (32/64-bit)
- MinGW-w64 (только для компиляции)
- Никаких дополнительных библиотек не требуется (всё статически скомпилировано)

## Компиляция

### Автоматическая компиляция (рекомендуется)

Просто запустите файл `compile.bat`:

```batch
compile.bat
```

Скрипт автоматически:
1. Проверит наличие MinGW в PATH
2. Скомпилирует все исходные файлы
3. Создаст исполняемый файл `build\FileSearchTool.exe`
4. Предложит запустить программу

### Ручная компиляция

Если у вас установлен MinGW-w64, выполните следующие команды:

```batch
# Компиляция ресурсов
windres resources\resource.rc -O coff -o build\resource.res

# Компиляция исходных файлов
gcc -c src\utils.c -o build\utils.o -Wall -O2 -std=c99
gcc -c src\search.c -o build\search.o -Wall -O2 -std=c99
gcc -c src\gui.c -o build\gui.o -Wall -O2 -std=c99
gcc -c src\main.c -o build\main.o -Wall -O2 -std=c99

# Линковка
gcc build\main.o build\gui.o build\search.o build\utils.o build\resource.res ^
    -o build\FileSearchTool.exe ^
    -mwindows -lcomctl32 -lgdi32 -lcomdlg32 -lshlwapi -lshell32
```

## Использование

1. **Запустите** `build\FileSearchTool.exe`
2. **Укажите путь** для поиска (или используйте кнопку "Обзор...")
3. **Введите ключевые слова** через запятую (например: `функция, переменная, массив`)
4. **Укажите расширения файлов** через запятую (например: `.txt, .c, .h, .cpp`)
5. **Настройте параметры поиска**:
   - ☑ Рекурсивный поиск — искать во всех подпапках
   - ☑ Учитывать регистр — различать заглавные и строчные буквы
   - ☐ Искать только целые слова — учитывать границы слов
6. **Нажмите "ПОИСК"**
7. Результаты отобразятся в области результатов с указанием файла и номера строки
8. При необходимости **экспортируйте результаты** в текстовый файл

## Технические детали

- **Язык**: C (стандарт C99)
- **GUI**: Win32 API
- **Компилятор**: MinGW-w64 GCC
- **Кодировка**: UTF-8 с поддержкой кириллицы
- **Размер exe**: ~100-200 KB

## Обработка ошибок

Программа обрабатывает следующие ошибки:
- Несуществующий путь для поиска
- Отсутствие ключевых слов
- Ошибки доступа к файлам/папкам
- Нехватка памяти

Ошибки доступа к файлам логируются, но не останавливают процесс поиска.

## Известные ограничения

- Максимальная длина строки результата: 1024 символа
- Максимальное количество ключевых слов: 50
- Максимальное количество расширений: 20
- Программа работает только на Windows
